# 🖼️ Image Gallery - Interactive Lightbox

A responsive image gallery with an interactive lightbox view, built with HTML, CSS, and JavaScript.

![Image Gallery Demo](image/image%201.jpeg)

---

## ✨ Features

- 🖼️ **Responsive Grid Layout** - Automatically adjusts to any screen size
- 🔍 **Lightbox View** - Click any image to view it in full screen
- ⬅️➡️ **Navigation** - Next/Previous buttons to browse images
- ⌨️ **Keyboard Support** - Use arrow keys (← →) and ESC to close
- 🎨 **Smooth Animations** - Hover effects and transitions
- 📱 **Mobile Friendly** - Works perfectly on all devices
- 🖱️ **Interactive Hover** - Scale effect on image hover
